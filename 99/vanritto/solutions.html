SOLUTIONS

The team was able to discover two workable solutions to the Problem Statement that would allow the ability to emulate the visualization of the multi-day bike fundraisier event route for the non-profit, The Liberty Project.

The two working solutions the team have demonstrated are strong solutions to the Problem Statement, creating an engaging interactive web map hosted on the webpage for The Liberty Project's fundraising event. Through these maps, the public will have an opportunity to engage with the fundraiser in a way that will allow them to visualize the event interactively, all while having access to the details about the fundraiser and the organization.

Making the Journey - Static Page to Animated Success

Discovering Storytelling

The first working solution was located while researching Mapbox tutorials. This tutorial provided unique Interactive Storytelling templates, also known as "Scrollytelling". This solution allows the team to generate a high-quality, interative story that will bring the public alongside the fundraising event through an immersive experience.

This template quickly became a top-favourite solution by the team as the template for the story content file (config.js) has the ability to move locations, determine settings for how the map and story are shown, and have text and images displayed for each new "chapter"

The Interactive Storytelling Maps are easy-to-generate with a free Mapbox account, access token and Github repository to host the story file. This made is a great option to explore to solve the Problem Statement.

Solution 2 - Animated Route Tracking

Discovering How to Live Update Features

Attempting to find a great solution for animating a path or line came from disovering animation of a route in a web map using the Mapbox GL JS tutorial, Update a Feature in Realtime. This tutorial provided framework where a feature that exists on a map can be updated. The tutorial example displays a path being drawn where the map view moves along side, following where the route is going - as though the data is occuring in real-time, like how Live Tracking would be displayed!

GeoJSON and Custom Animation

Exploring the tutorial provided a method to examine the working of basic line animation using a GeoJSON, but the team had yet to sucessfully impliment working GeoJSON of our own. Some experimentation evolved into the map below, with modified base mapping, and line colouring, and ideas on how to manipulate timing and view in the web map

Figuring Out Animation

Trial and error finally yields positive results when the team loads a working GeoJSOn for the first time. Experimenting with options like preparing line files in google maps and ESRI ArcGIS Online led to the discovery that a lot of non-native GeoJSON files would not load into Mapbox without manipulation of the files to remove the CRS information. The example in the demonstration below shows a line feature created in ArcGIS Pro, converted to GeoJSON in ArcGIS Online, and manipulated to remove the CRS information causing a load error in Mapbox. The team discovered we could host the GeoJSON right in the HTML and bypass keeping it in Mapbox.

The GeoJson continued to prove problematic and required more investigation.

The Final Solution

After understanding the shortcomings of the modified GeoJson, the Team went to Mapbox and created a new modified line feature, with an intermediate number of points to roughly resemble to proposed bike route for the project. Some quick alterations to the line colour and opacity, and loading a custom map style, also generated in Mapbox. We finally reached a reasonable success. We now know that we need to refine the GeoJSON to be an in-between number of points from too few (which results in faster traversal of the line segment and jerky camera motion) and too many, which results in very slow but smooth movements, like we saw in the initial demonstration templates. Manipulation of the interval timer will also be crucial for constructing a web map that has smooth animation, draws more slowly, properly follows the designated route along with roads.
