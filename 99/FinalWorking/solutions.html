SOLUTIONS

The team was able to discover two workable solutions to the Problem Statement that would allow the ability to emulate the visualization of the multi-day bike fundraisier event route for the non-profit, The Liberty Project.

OPTION 1 FOR FORMAT

Solution 1 - Interactive Storytelling

Discovering Storytelling

The first working solution was found through researching Mapbox tutorials where one was found that provides unique Interactive Storytelling templates, also known as "Scrollytelling". This solution allowed the team to generate a high-quality, interative story that will bring the public alongside the fundraising event through an immersive experience.

This template quickly became a top-favourite solution by the team as the template for the story content file (config.js) has the ability to move locations, determine settings for how the map and short are shown, and have text and images displayed for each new "chapter"

The Interactive Storytelling Maps are easy-to-generate with a free Mapbox account, access token and Github respository to host the story file.

Solution 2 - Animated Route Tracking Solution

Tutorial

The second working solution that the team came up with involved the animation of a route in a web map using the Mapbox GL JS tutorial, Update a Feature in Realtime. This tutorial provided framework where a feature that exists on a map can be updated. The tutorial example displays a path being drawn where the map view moves along side, following where the route is going - as though the data is occuring in real-time, like how Live Tracking would be displayed!

Further experimentation by the team, lead to exploring working with a GeoJSON layer representing the route layer with a web map.
