Tutorials Shed Light

Exploring animation began with the Mapbox GL JS tutorial "Update a feature in realtime". This tutorial provided the framework for furthur experiments when it explored working with GeoJSON data in a web map.

Once we understood HOW to make the animation, we began to experiment with templates and variations of GeoJSON files.

A final solution begins to emerge with furthur refinement of a custom GeoJSON and map styles applied, both made Mapbox. Armed with this information, we can refine the route model and code to better reflect the path, and create a more user freindly viewing experience.
